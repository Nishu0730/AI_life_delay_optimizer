# AI Life Delay Optimizer 🚀

An AI-powered productivity and task management application built using Flask, JavaScript, HTML, CSS, and SQLite.

## Features

- 🔐 User Authentication (Login & Signup)
- 👤 User Profile
- ➕ Add Tasks
- ✏️ Edit Tasks
- 🗑 Delete Tasks
- ✅ Mark Tasks as Completed
- 📊 Analytics Dashboard
- 🧠 AI Insights
- 💡 AI Recommendations
- 🔔 Success Notifications
- 📱 Mobile Responsive Design
- 🌙 Dark/Light Mode
- 📈 Productivity Score Tracking

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- Flask

### Database
- SQLite

## Installation

```bash
git clone <your-repository-url>

cd AI_life_delay_optimizer

pip install flask werkzeug

python app.py
```

Open:

http://127.0.0.1:5000
